## VARIABLES D'ENVIRONNEMENT n8n à configurer

Ajoutez ces variables dans **Settings → Environment Variables** de votre n8n :

| Variable | Description | Où l'obtenir |
|----------|-------------|--------------|
| `ANTHROPIC_API_KEY` | Clé API Claude | console.anthropic.com |
| `FACEBOOK_PAGE_ID` | ID de votre page Facebook | Page Facebook → À propos |
| `FACEBOOK_PAGE_ACCESS_TOKEN` | Token d'accès page (long-lived) | developers.facebook.com |
| `INSTAGRAM_ACCOUNT_ID` | ID compte Instagram Business | Graph API Explorer |
| `DEFAULT_IMAGE_URL` | URL d'une image par défaut publique | Hébergez sur votre site |
| `LINKEDIN_ACCESS_TOKEN` | Token OAuth LinkedIn | LinkedIn Developer Portal |
| `LINKEDIN_ORG_ID` | ID de votre organisation LinkedIn | URL de votre page LinkedIn |
| `NOTIFICATION_EMAIL` | Email pour recevoir scripts TikTok/Snap | Votre email |

## ÉTAPES D'INSTALLATION

### 1. Importer le workflow
- Ouvrez votre n8n
- Menu → **Workflows → Import from file**
- Sélectionnez `n8n_ship_cars_social.json`

### 2. Configurer les credentials
- **Facebook/Instagram** : créez une app sur developers.facebook.com, activez les permissions `pages_manage_posts`, `instagram_basic`, `instagram_content_publish`
- **LinkedIn** : créez une app sur linkedin.com/developers, activez `w_member_social` ou `w_organization_social`
- **Gmail** (pour TikTok/Snap) : ajoutez un credential Gmail dans n8n

### 3. Configurer les variables d'environnement
- n8n → Settings → Environment Variables
- Ajoutez toutes les variables du tableau ci-dessus

### 4. Tester
- Ouvrez le workflow importé
- Cliquez **"Test workflow"** sur le nœud Schedule
- Vérifiez que le contenu est généré correctement avant d'activer

### 5. Activer
- Cliquez le bouton **Active** en haut à droite du workflow
- Les posts partiront automatiquement lundi, mercredi et vendredi à 10h

## NOTES IMPORTANTES

- **TikTok & Snapchat** : ces plateformes ont des APIs très restrictives (TikTok API Creator nécessite une approbation). Pour l'instant, le workflow envoie les scripts par email pour publication manuelle.
- **Images** : pour Instagram, vous devez fournir une URL d'image publique. Pensez à créer un dossier `/public/social/` sur votre site avec des photos de vos véhicules.
- **Instagram doit être un compte Business** connecté à votre page Facebook pour fonctionner avec l'API Graph.
- **Fréquence** : 3 posts/semaine est une bonne cadence. Vous pouvez modifier le cron dans le nœud Schedule (`0 10 * * 1,3,5` = Lun/Mer/Ven à 10h).
